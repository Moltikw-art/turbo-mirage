# Трекер расходов (Expense Tracker)

**Автор:** Маткин Михаил Сергеевич

---

## Описание

Десктопное GUI-приложение для учёта личных расходов.
Программа позволяет добавлять, фильтровать, удалять расходы и сохранять их в JSON-файл.

**Важная особенность:** использует **только стандартную библиотеку Python** (Tkinter, json, datetime, os).
Не требует установки дополнительных пакетов.

---

## Функциональные возможности

- ✅ Добавление расходов (сумма, категория, дата)
- ✅ Фильтрация по категории
- ✅ Фильтрация по диапазону дат
- ✅ Подсчёт общей суммы за выбранный период
- ✅ Удаление записей
- ✅ Автоматическое сохранение в JSON
- ✅ Автоматическая загрузка при запуске
- ✅ Валидация ввода (сумма > 0, правильный формат даты)

---

- Никаких дополнительных библиотек не требуется
